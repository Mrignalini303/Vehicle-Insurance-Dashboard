# Vehicle-Insurance-Dashboard
**1. Project Description and Purpose**
The Dashboard is about a vehicle insurance company named "EIC". The analysis is being done to determine whether or not the company is making a profit. What are the KPI affecting loss, and why?

**2. Project findings**
The company has been in the loss since the start. The policy-to-claim ratio has always been less than 100%, indicating that claims have always exceeded the policy bought. 
Vehicle type: Truck is the most claimed vehicle, resulting in a heavy loss for the company.
The agricultural and farming industry, where the vehicle claims have been the least. This can be a very good reason to provide more policies in this industry.
